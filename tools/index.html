下面是我平时开发软件项目常用的工具, 都是十分优秀的生产力工具.

设计类工具

  • ArgoUML UML 设计软件
  • Pencil Project 原型设计工具
  • MockPlus 原型设计工具(收费)
  • Xmind Mind map工具
  • DrawIO 通用diagram工具
  • excalidraw 手绘类diagram工具

开发工具

  • Visual Studio Code 用来写web前端代码
  • SpringToolSuite 开发Java项目
  • Visual Studio 社区版 开发C#项目
  • Pycharm 社区版 开发Python项目
  • HBuilderX 用来写web前端代码

代码管理工具

  • WinMerge 代码差异对比工具
  • Git Extensions git GUI 工具

数据库工具

  • PDManer 数据模型设计工具
  • Deaver 数据库客户端

部署工具

  • Filezilla FTP 客户端
  • Putty SSH 客户端

项目管理

  • GanttProject 甘特图工具
